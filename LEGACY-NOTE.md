# Legacy snapshot — Livewire prototype (May 2025)

This branch is a **non-git file snapshot** of the original Cinexio prototype,
built with **Laravel Breeze + Livewire** before the project was rewritten on the
current Vue/Inertia stack (see `main`).

It is kept for reference only — it carries no commit history and is **not maintained**.
Notable here that differs from `main`: the original May-2025 schema and the early
`Audio` and `Wishlist` models.
